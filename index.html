Python flashcard

Python là gì?

Python là một ngôn ngữ lập trình bậc cao, đa năng, được thiết kế để dễ đọc, dễ viết và dễ bảo trì.

Cách khai báo biến trong Python như thế nào?

Biến được khai báo bằng cách gán giá trị trực lùi
ví dụ: x = 5, name = "Python".

List trong Python là gì?

List là một cấu trúc dữ liệu cho phép lưu trữ một danh sách các phần tử có thể thay đổi
ví dụ: my_list = [1, 2, 3, "Python"].

Vòng lặp for trong Python được sử dụng như thế nào?

Vòng lặp for dùng để lặp qua các phần tử của một dãy
ví dụ: for i in range(5): print(i).

Hàm len() trong Python có tác dụng gì?

Hàm len() trả về độ dài (số phần tử) của một đối tượng như list, string, tuple.

Lambda function trong Python là gì?

Lambda function là một hàm ẩn danh có thể có nhiều tham số nhưng chỉ có một biểu thức
ví dụ: lambda x, y: x + y.

Dictionary trong Python là gì?

Dictionary là một cấu trúc dữ liệu lưu trữ các cặp khóa-giá trị
ví dụ: my_dict = {"name": "Python", "version": 3.9}.

Làm thế nào để đọc một file trong Python?

Sử dụng hàm open()
ví dụ: with open("file.txt", "r") as file: content = file.read().

Làm thế nào để xử lý ngoại lệ trong Python?

Sử dụng khối lệnh try và except

Thư viện numpy trong Python được sử dụng để làm gì?

Thư viện numpy được sử dụng để tính toán khoa học, đặc biệt là với các mảng và ma trận đa chiều.

Javacript flashcard

Tiến độ: 0%

JavaScript là gì?

JavaScript là một ngôn ngữ lập trình được sử dụng để tạo ra các trang web tương tác. Nó là một phần quan trọng của các công nghệ web cùng với HTML và CSS.

Sự khác biệt giữa let, var và const trong JavaScript là gì?

var có phạm vi toàn cục hoặc hàm, let và const có phạm vi khối. const được sử dụng để khai báo các hằng số không thể thay đổi.

Một callback function là gì?

Callback function là một hàm được truyền vào một hàm khác như là một đối số và được thực thi sau khi hàm đó hoàn thành.

Arrow function trong JavaScript là gì?

Arrow function là một cú pháp ngắn gọn để viết hàm trong JavaScript, với cú pháp:
const functionName = (parameters) => { // code }.

DOM là gì?

DOM (Document Object Model) là một giao diện lập trình cho các tài liệu HTML và XML, đại diện cho trang dưới dạng cấu trúc cây, cho phép thay đổi nội dung và cấu trúc trang web.

Sự khác biệt giữa == và === trong JavaScript là gì?

== so sánh hai giá trị sau khi chuyển đổi kiểu của chúng nếu cần thiết (so sánh lỏng lẻo), còn === so sánh cả giá trị và kiểu dữ liệu (so sánh chặt chẽ).

Event delegation là gì?

Event delegation là kỹ thuật xử lý sự kiện ở một phần tử cấp cao hơn thay vì phần tử cụ thể nơi sự kiện xảy ra, lợi dụng sự nổi bọt (bubbling) của sự kiện.

Async/Await là gì?

Async/Await là cú pháp trong JavaScript để xử lý các thao tác bất đồng bộ một cách đồng bộ, giúp mã dễ đọc và dễ bảo trì hơn.

Promise trong JavaScript là gì?

Promise là một đối tượng đại diện cho kết quả của một thao tác bất đồng bộ, có thể ở trạng thái chờ (pending), thành công (fulfilled), hoặc thất bại (rejected).

Một Closure trong JavaScript là gì?

Closure là một hàm có quyền truy cập vào các biến từ phạm vi hàm bao quanh nó, ngay cả sau khi hàm bao quanh đã thực thi xong.

HTML flashcard

HTML là gì?

HTML (HyperText Markup Language) là ngôn ngữ dùng để tạo và cấu trúc các trang web.

Thẻ <header> dùng để làm gì?

Thẻ <header> chứa các thông tin meta, liên kết đến CSS, tập lệnh và tiêu đề trang.

Thẻ <a> có chức năng gì?

Thẻ <a> được sử dụng để tạo liên kết đến các trang web khác.

Thẻ <img> dùng để làm gì?

Thẻ <img> được dùng để chèn hình ảnh vào trang web.

Thẻ <div> có công dụng gì?

Thẻ <div> được dùng để tạo các khối chứa nội dung và dễ dàng quản lý bố cục trang.

Thẻ <form> có chức năng gì?

Thẻ <form> dùng để tạo biểu mẫu thu thập dữ liệu từ người dùng.

Sự khác nhau giữa thẻ <ul> và <ol> là gì?

Thẻ <ul> tạo danh sách không thứ tự (bullet points), còn thẻ <ol> tạo danh sách có thứ tự (số thứ tự).

Thẻ <table> dùng để làm gì?

Thẻ <table> dùng để tạo bảng dữ liệu trên trang web.

Thuộc tính 'alt' của thẻ <img> có tác dụng gì?

Thuộc tính 'alt' cung cấp văn bản thay thế cho hình ảnh, hỗ trợ SEO và truy cập.

Thẻ <meta> dùng để làm gì?

Thẻ <meta> cung cấp thông tin meta như mô tả, từ khóa, và thông tin tác giả cho trang web.

CSS flashcard

Tiến độ: 0%

CSS là gì?

CSS (Cascading Style Sheets) là ngôn ngữ dùng để mô tả giao diện và định dạng của một trang web được viết bằng HTML

Selector trong CSS là gì?

Selector là một thành phần trong CSS dùng để chọn các phần tử HTML mà bạn muốn áp dụng kiểu dáng.

Cách viết một class selector trong CSS?

Sử dụng dấu chấm trước tên class
ví dụ: .myClass { color: red; }.

ID selector được viết như thế nào?

ĐSử dụng dấu thăng trước tên ID,
ví dụ: #myId { font-size: 20px; }.

Inline CSS là gì?

Inline CSS là cách thêm kiểu dáng trực tiếp vào thẻ HTML thông qua thuộc tính style
ví dụ: <div style="color: blue;">.

Sự khác nhau giữa margin và padding?

Margin là khoảng cách bên ngoài viền của phần tử, còn padding là khoảng cách bên trong viền của phần tử.

Cách dùng thuộc tính flex?

display: flex; sử dụng để tạo layout linh hoạt trong CSS, sắp xếp các phần tử con một cách dễ dàng.

Chức năng của media queries là gì?

Media queries giúp tạo ra các kiểu dáng CSS khác nhau cho các kích thước màn hình khác nhau.

Pseudo-class là gì?

Pseudo-class là từ khóa được thêm vào selector để chỉ định trạng thái đặc biệt của phần tử
ví dụ: :hover, :active.

Cách liên kết tệp CSS ngoài vào trang HTML?

Sử dụng thẻ <link> trong phần <head>, ví dụ: <link rel="stylesheet" href="styles.css" >.
